# UI/UX Design Resources

## Concept

UI/UXデザイナーとWeb制作者向けの実践的技術リソース集です。マルチレゾリューションや煩雑な更新や修正に耐えうる非破壊編集やプロシージャルなワークフローを構築し、今後の生成AIが主導するさらなる自動化プロセスに対応したモダンなUI設計環境を研究しています。
